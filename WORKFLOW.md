# ARUNEE × CHATGPT — WORKFLOW RULES (Source of Truth)

เอกสารนี้คือ “กติกาการทำงาน” ระหว่างผู้ใช้ (เจ้าของโปรเจกต์) และ ChatGPT (ผู้ช่วย)
ใช้เป็นกฎหมายสูงสุดของงานนี้ เพื่อป้องกันการทำงานซ้ำซ้อน และกันการทับโค้ดส่วนที่โอเคแล้ว

---

## 1) Process (ห้ามข้ามขั้น)

1) ผู้ใช้บรีฟความต้องการ
2) ChatGPT ต้องสรุปสิ่งที่เข้าใจ + ระบุให้ชัด:
   - จะไปแก้ไฟล์ไหนบ้าง
   - จะเพิ่มไฟล์ใหม่อะไรบ้าง (ถ้ามี)
   - ฟีเจอร์/ส่วนไหน “จะไม่แตะ”
   - ความเสี่ยง/ผลกระทบที่อาจเกิด
3) ChatGPT ขอ “อนุมัติ”
4) ผู้ใช้พิมพ์ว่า “อนุมัติ”
5) ChatGPT เริ่มเขียนโค้ดตามที่สรุปไว้เท่านั้น

---

## 2) Default Rule: Replace Whole File

ค่าเริ่มต้นของการส่งโค้ดคือ **“โยนทับทั้งไฟล์”** (Replace Whole File)
เพื่อให้ผู้ใช้เอาไป replace ใน GitHub ได้ง่ายและไม่พลาด

ข้อยกเว้น:
- ถ้าเป็นการ “จูนตัวเลขเล็ก ๆ” ที่ต้องดูผลทันที
  → ใช้วิธี “แก้เฉพาะค่า/เฉพาะบรรทัด” แทนการทับทั้งไฟล์

---

## 3) Do Not Touch Unmentioned Sections

**ห้ามแตะ** ฟังก์ชัน/section/โค้ดที่:
- ผู้ใช้ไม่ได้พูดถึงในรอบนั้น
- หรือส่วนที่ “โอเคแล้ว” และใช้งานได้แล้ว

ถ้าต้องเพิ่มความสามารถใหม่:
- ต้อง “แยกเป็นไฟล์ใหม่” ตามหลัก Separation of Concerns
- ยกเว้นผู้ใช้สั่งชัดเจนว่า
  **“แก้ไฟล์เดียวเพื่อจูนบนมือถือ”** หรือ **“แก้ไฟล์เดียว”**

---

## 4) Repo & Deploy Responsibilities

- **Base files = ไฟล์ใน GitHub repo (branch main) เท่านั้น**
- ผู้ใช้เป็นคน:
  - upload / replace ไฟล์ใน GitHub
  - ปล่อยให้ Vercel auto-deploy

ChatGPT เป็นคน:
- เขียนโค้ดให้ตาม workflow
- ส่งโค้ดแบบพร้อม replace
- บอกตำแหน่งไฟล์/ชื่อไฟล์ให้ชัดทุกครั้ง

---

## 5) When Starting a New Chat

ทุกครั้งที่เริ่มแชทใหม่ ผู้ใช้จะ paste “Template เปิดแชท” เพื่อ sync กติกาและสถานะล่าสุด

**Template เปิดแชท**
```text
LOAD WORKFLOW: ARUNEE

- ใช้ Workflow rules เดิมทั้งหมด
- อย่าทับฟังก์ชัน/section ที่ไม่ได้พูดถึง
- เพิ่มอะไรให้แยกไฟล์ (ยกเว้นสั่งชัดว่าแก้ไฟล์เดียวเพื่อจูน)
- Base = ไฟล์ใน GitHub repo (main)
- มึงเป็นคน replace/upload → Vercel auto-deploy
- ขออนุมัติก่อนเขียนโค้ดทุกครั้ง
- ถ้าจูนเลขเล็ก ๆ ขอคุยแบบ “แก้เฉพาะค่า”

สถานะล่าสุด:
- (พิมพ์สั้น ๆ ว่าตอนนี้ติดอะไร/อยากได้อะไร)
